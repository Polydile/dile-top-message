# \<dile-top-message>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i dile-top-message
```

## Usage
```html
<script type="module">
  import 'dile-top-message/dile-top-message.js';
</script>

<dile-top-message></dile-top-message>
```
